About the Game

1
What it is: a fast, grid‑based word puzzle. Pick any letters, submit valid words, and clear the board before stacks overflow.
2
How it plays: pick any tiles (no adjacency). Minimum word length is 3. Submitting clears the top letter from each selected stack. New letters spawn on a timer; a dashed outline warns the next spawn.
3
Difficulty & tempo: Easy 10s ×1 • Medium 7s ×1 • Hard 5s ×2 • Insane 4s ×3 (short words may bump to 4). Word length affects tempo: 3 → speeds up, 4 → normal, 5+ → slows down.
4
Settings: Grid size 4–10 (default 6). Optional lose threshold (Off / 5 / 7 / 9). Words are validated online via dictionaryapi.dev.
